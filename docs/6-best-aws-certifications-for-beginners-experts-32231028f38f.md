除非你想成为 AWS 英雄，否则你不必通过所有这些 AWS 认证。你应该选择一个最适合你的个人资料，职业抱负和经验。所以，事不宜迟，让我们继续关注你在 2022 年可以瞄准的顶级 AWS 和云认证。
