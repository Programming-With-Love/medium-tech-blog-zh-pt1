# 什么是运营商？

运算符用于比较值、执行算术运算等。比如我们拿一个简单的表达式来说，4 + 5 等于 9。这里的 4 和 5 称为操作数，而“+”称为运算符。JavaScript 由不同类型的操作符组成，用于执行不同的操作。
