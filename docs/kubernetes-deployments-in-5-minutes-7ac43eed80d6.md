简单地说，Kubernetes 部署负责创建、更新和删除应用程序的实例(pods)。它还为*提供了一种自我修复机制*，在你的应用崩溃时帮助你。在此之前，如果你还不知道，Kubernetes 是一个开源的容器编排平台，可以运行和管理企业级的 web 可伸缩应用程序。
