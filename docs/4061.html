虚伪的对象(扒手)

一位C#开发者说，小心扒手。不幸的是为时已晚…这是为新手准备的。

遇到假物体。这些流氓总是找麻烦。

- empty string
- the number zero
- null
- NaN
- the boolean false
- undefined

再看看他们。他们会让你检查你的钱包几次。看看这些，告诉我你以前没有爱上它们。

带回家。对falsy对象使用否定，但保留==，===，！=, !==其余部分。

Falsy值并不总是等于false，但是当与negation一起使用时，所有的falsy值都计算为true。
