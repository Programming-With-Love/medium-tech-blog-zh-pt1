欢迎来到Android中的Now，这是您对Android开发世界中新的和值得注意的事物的持续指导。随着2021年接近尾声，让我们反思一下今年我们在Android中做的一些大事。

安卓12

我们发布了Android 12 并推送到 Android开源项目 (AOSP)。我们引入了一种新的设计语言，叫做材料你。我们减少了核心系统服务使用的CPU时间，增加了性能类设备功能，并增加了新功能来提高性能。通过隐私仪表盘和其他新的安全和隐私功能，用户可以更好地控制自己的隐私。我们通过一个用于丰富内容插入的统一API、兼容媒体转码、更简单的模糊和效果、 AVIF图像支持、增强的触觉、新的相机效果/功能、改进的本机崩溃调试、对圆角的支持、下载时播放，以及

Android 12L 也在测试中，这让Android 12在大屏幕上更加出色。它包括使多任务处理更加直观的工具，包含对兼容性模式的改进等等！今天就来试试。

构成

Jetpack Compose ，Android的现代原生UI工具包变得稳定并准备好在生产中采用。它与您现有的应用程序互操作，与现有的Jetpack库集成，使用直接的主题化实现材质设计，使用最少的样板文件支持带有惰性组件的列表，并且拥有一个强大的、可扩展的动画系统。您可以在 Compose学习路径中了解更多关于使用Compose的信息，并在 Compose路线图中了解我们在未来的Compose版本中的发展方向。

培养

今年，Android培训团队在Kotlin 发布了 Android基础的最后四个新单元。

  • 第3单元讲述了导航，在这里你可以学习如何在应用程序的不同屏幕之间来回导航。
  • 单元4包括连接互联网。您将为异步代码编写协程，了解HTTP和REST以从互联网获取数据，并使用Coil库在您的应用程序中显示图像。
  • 第5单元讲述了数据持久性，在这里您将学习如何在设备上存储数据，这是确保流畅一致的用户体验的一部分，即使在网络中断时也是如此..
  • 第6单元讲述了工作管理器，你可以使用Android Jetpack的工作管理器API来安排后台工作，即使应用程序退出或设备重启，后台工作也会继续运行。

疯狂的技能

对于MAD Skills，我们度过了令人惊叹的一年，发布了多个视频和博客系列，涵盖了许多重要主题:

  • Kotlin和Jetpack —了解Jetpack KTX库的基础知识，如何使用协程和流简化回调，以及如何使用和测试Room/work manager API。
  • Motion Layout —了解如何使用Motion Layout及其设计工具来创建丰富的动画体验。
  • 工作管理器 —了解如何使用工作管理器安排关键后台工作:从基本使用、线程、定制配置等。
  • 导航 —了解导航组件的基础知识、工具的特定功能以及创建和导航到目的地的API。
  • 性能 —了解如何使用系统跟踪和采样分析来调试应用中的性能问题。
  • Hilt —了解如何在您的Android应用中添加和使用Hilt进行依赖注入，使用Hilt进行测试的最佳实践，以及更多高级内容。
  • 分页 —学习分页的基础知识，从核心类型到将它们绑定到UI元素。
  • Gradle —了解如何配置您的构建，根据您的需求定制构建过程，以及如何编写自己的插件来进一步扩展您的构建。
