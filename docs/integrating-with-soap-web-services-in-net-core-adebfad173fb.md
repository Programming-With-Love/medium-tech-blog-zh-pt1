SOAP(简单对象访问协议)是一种消息协议规范，通过使用 XML 信息集作为其消息格式，在 web 服务的实现中交换结构化信息(来源:[https://en.wikipedia.org/wiki/SOAP](https://en.wikipedia.org/wiki/SOAP))。消息可以通过各种应用层协议，例如 HTTP(s)。
