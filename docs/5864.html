  1. 去https://www.oracle.com/cloud/free的。
  2. 点击“免费开始”。
  3. 填写表格并创建一个帐户。注意:有些地区不具备所有功能。特性矩阵(这听起来像是来自 变形金刚电影 ) 显示了每个地区所有可用特性的列表。
  4. 创建您的帐户后，登录并转到仪表板。

注意:注册时你需要一张信用卡，但是除非你升级你的免费等级账户，否则不会被扣费。您必须将您的帐户更改为付费帐户才能收费。
