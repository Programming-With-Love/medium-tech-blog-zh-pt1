## 你有没有想过匕首在引擎盖下是做什么的🤔？

每个注释都有一个*点*🔍！查看这些备忘单，了解 Dagger 如何解释它们以及它为您生成的代码。关于生成的代码和 Dagger 注释的解释可以在备忘单中作为注释找到。
