调试在软件开发中非常常见，知道如何调试 NodeJS 应用程序非常重要。当谈到 NodeJS 应用程序时，我们可以使用 Chrome DevTools 调试该应用程序。您需要在 inspect 模式下启动应用程序，以便该过程在可调试状态下启动。
