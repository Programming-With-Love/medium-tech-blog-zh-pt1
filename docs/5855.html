我近年来遇到的最令人兴奋的框架之一是Dapr ( dapr.io ) — d 分布式APapplicationruntime。Dapr是一个跨技术的运行时框架，支持多种语言的应用程序。

Dapr文档描述了如何在本地和Kubernetes集群上安装Dapr。该文档明确链接到Azure、AWS和GCP文档，但没有提到OCI上的OKE。所以在这篇文章中，我将向你展示如何在OKE上使用Dapr。
