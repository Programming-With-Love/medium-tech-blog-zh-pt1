如何在Python中求一个数的倒数？

很简单！您可以编写一个Python程序，它接受一个输入数字，并对其进行反向处理。整数值存储在使用条件检查的变量中，然后数字的每个数字存储在另一个变量中，该变量将打印反转的数字。在Python中，可以使用不同的方法来反转数字，让我们来看看实现相同功能的Python程序。

Python程序反转一个数字

在Python编程语言中，有两种方法可以反转一个数字

  • 使用循环
  • 使用递归

使用循环反转数字

# Get the number from user manually

使用递归反转一个数

# Python Program to Reverse a Number using RecursionNum = int(input("Please Enter any Number: "))Result = 0
def Result_Int(Num):
    global Result
    if(Num > 0):
        Reminder = Num %10
        Result = (Result *10) + Reminder
        Result_Int(Num //10)
   return Result
Result = Result_Int(Num)
print("n Reverse of entered number is = %d" %Result)
