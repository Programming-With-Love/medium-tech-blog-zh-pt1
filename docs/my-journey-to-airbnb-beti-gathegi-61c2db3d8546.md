从探索各大洲的职业生涯到现在帮助他人在 Airbnb 找到自己的位置。

在尝试了从电视制作到大学传播和营销的一系列职业后， [*贝蒂·盖特吉*](https://www.linkedin.com/in/betigathegi) *在 Airbnb 的 TechED(技术教育)团队担任高级项目经理。当她不在#bookworms Airbnb Slack 频道潜伏时，你可以找到 Beti 领导的 Bootcamp，这是我们为新技术员工提供的入职计划，带领工程师和数据科学家完成他们在 Airbnb 的第一次承诺。在担任这一职务之前，贝蒂是 Connect 的招聘项目经理，Connect 是 Airbnb 的工程学徒计划，针对的是非传统技术背景的人。*

*贝蒂本人有着非传统的背景，她拥有新闻学位和科技行业以外的一些经验，包括在国外的大量时间。她是多样性和包容性的主要倡导者；她领导 Bootcamp 的部分职责包括建立公司文化，鼓励新员工以自己独特的方式塑造公司文化。*

# 设定我自己的方向

我把自己描述成半个东海岸，半个西海岸，外加一点海外时间。我是肯尼亚移民的孩子，在加州旧金山湾区一个叫奥尔巴尼的小镇长大。当我 15 岁的时候，我搬到了东海岸，过了很多年我才发现自己又回到了海湾地区。

很长一段时间，我想成为一名记者。为此，我在大学里学习了新闻学，作为我传播学学位的一部分。我从来没有执着于一条特定的道路，当然也探索了很多才达到我现在的位置。我的父亲在 40 岁左右获得了法律学位，他是我乐于尝试新事物的指路明灯。我发现个人探索是一种解放——创造我自己的、有机的道路给了我一个机会去发现我的好恶，以及我的技能和成长机会。

我的人生观的一部分是，阻止一些不适合自己的事情是可以的。有时候会有很多惯性让你很难停下来改变方向，但我认为做出追求另一条道路的决定真的很勇敢，可以作为荣誉的徽章佩戴。就我而言，我开始攻读文科硕士学位，研究南亚移民，尤其是印度移民的子女。受到其他人故事的启发，我渴望更多地了解自己的背景和历史。我选择离开我的项目去肯尼亚生活，亲身体验肯尼亚文化。在那之前，我只和家人去过肯尼亚，所以这是我自己看这个国家的一个新镜头。

# 在三大洲生活和工作

在肯尼亚生活是一次变革性的经历，帮助我更深刻地理解了自己的身份。以前有人告诉我，我不够肯尼亚人或不够美国人，实际上在肯尼亚生活并遇到各种各样的人使我意识到没有单一的方式成为肯尼亚人，就像没有单一的方式成为美国人或任何文化一样。在国外期间，我也意识到我已经准备好获得更多的实践经验，并进入工作世界。

准备过冒险的生活，我搬到了纽约市，但无意中陷入了金融危机，似乎每个人都被解雇了。我在零售业工作了一段时间，但在纽约并没有持续太久。这只是一系列新经历中的第一个，我的下一个经历是在一家电视制作公司做助理，直到今天，我还在 IMDB 获得了《国家地理》节目*沼泽人*的四集学分。如果这还不够的话，我还为尼尔森写电视问答比赛，为南佛罗里达大学做市场营销，在澳大利亚土著艺术画廊工作。

最终，当一个朋友招募我加入 Lyft 担任客户支持角色时，我碰巧进入了科技行业。这对我来说是一个全新的世界，我抓住每一个机会参与进来，并把我的技能应用到一家成长中的公司。几乎是出于偶然，我通过公司内部电子邮件名单召集员工的倡议变成了员工资源小组或 ERGs。我帮助组建了 Black ERG 和妇女午餐会，同时也支持其他想要为他们的社区创造类似空间的人。很自然地，我在多样性和归属感的对话中发挥了重要作用，并确保教育自己成为这些讨论中有思想的贡献者。后来，这变成了一份专注于 Lyft 文化的正式工作，后来我搬到了潘多拉，担任多元化和归属感项目经理。

# 加入 Airbnb

当我申请加入 Airbnb 时，我已经转而负责 Pandora 的大学招聘项目。我注意到学生有巨大的潜力，并发现在这个关键的管道上工作真的很有影响力。也就是说，从大学直接进入科技行业并不是唯一的途径，我自己的职业生涯就是一个最好的例子。我欣然接受了加入 Airbnb 的机会，成为一名学徒项目经理，在那里我有机会为具有非常规背景的工程师改善加入公司的途径。

我负责学徒项目两年了，虽然这个角色主要是在招聘方面，但我们与工程部门密切合作，我开始对 TechED 产生兴趣，我现在所在的团队负责 Airbnb 技术人员的入职流程。我已经定期与我的经理 Leo 会面，讨论我的职业发展，所以我开始分享我在那种情况下的目标。Leo 非常支持我们，他是 Airbnb 文化的缩影，这种文化让人们能够探索他们的激情和激发他们的动力。

我联系了 TechED 团队的经理，表达了我对这个领域的兴趣，不久之后，一个职位正好空缺。面试后，我得到了目前的职位，负责 Airbnb 的 Bootcamp 项目，这是所有软件工程师和数据科学家在公司的第一周都要经历的入职流程。我经历了很多其他的角色，才达到现在的位置，但这伴随着不可估量的学习，我不会有任何其他的方式！在经历了这么多之后，我觉得自己有独一无二的能力来欢迎人们接受新的体验或挑战。

# 带领训练营和其他人加入 Airbnb

我目前的角色与我帮助人们适应环境并为他们提供最终获得成功的资源的热情一致。倡导令人难以置信的新员工体验，帮助新团队成员对他们在 Airbnb 的各自角色充满信心，并支持他们做出第一次承诺或代码更改，这是令人满意的。我努力尊重人们的时间，让训练营尽可能对每个参与者有意义、有吸引力、有价值。

入职是一项具有挑战性的任务，因为有多个变量。通常情况下，您会让不同角色、不同级别的人员加入不同的团队，这些团队可能会使用他们自己的工具和流程。在提供一般信息和高度相关但也高度具体的信息之间需要平衡。远程入职也增加了自身的挑战。

也就是说，我喜欢共同创造解决方案。我开始与工程和数据科学团队中非常聪明的人一起工作，以确定和阐明我们的挑战，研讨想法，执行解决方案，监控进展，并进行迭代。我从这个过程和我们的合作中获得了巨大的能量。

# 工作的第一周如何能留下持久的影响

我也非常感谢与我合作的许多志愿者，他们为我们的技术人员创造了入职体验，并为他们的成功做好了准备。例如，我们将每个新员工与他们团队中的一个伙伴配对。他们既要界定员工的初始项目，也要回答入职时不可避免会出现的许多问题。

我们有来自不同团队的志愿者，他们举起手来为每个新员工举办训练营和领导会议，他们中的大多数人都有归属感。此外，整个行业有一个伟大的合作者社区，可以进行基准测试并获得指导，因为入职是许多公司都在努力解决的一个具有挑战性的问题。

除了入职的技术部分，Bootcamp 在设置 Airbnb 的文化方面发挥着关键作用。尤其是在远程工作环境中，入职培训的质量会决定新员工是否能感受到社区感，是否能自在地参与其中。我们强调归属感和包容性是我们文化的核心价值观，我们欢迎新员工带着自己的特殊品质融入我们不断发展的文化。
