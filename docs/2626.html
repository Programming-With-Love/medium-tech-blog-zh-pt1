要实现窗口小部件的变形，添加阴影是必须的。要在 小部件 上创建阴影，常见的方法是将其包裹在 容器 中。我们需要box shadowfrombox decoration属性。而提供box decoration的小部件之一就是 容器 小部件。所以基本上，在这个方法中，我们将通过把它包装在一个 容器 中来给 小部件 添加神经变形功能。

属性:

  • 双 borderRadius :创建圆角的方框。
  • 偏移 offset :创建一个偏移。第一个参数设置水平分量dx，第二个参数设置垂直分量dy。
  • doubleblurRadius :高斯与阴影形状卷积的标准差。应用于box shadow属性。
  • 颜色 topShadowColor:出现在左上角的阴影颜色。
  • 颜色 bottomShadowColor:出现在右下角的阴影颜色。

我只是在左上角和右下角应用阴影。您还可以自定义该方法，这样您就可以通过在列表中添加一个box shadow并设置左下方的 Offset ( —，+)和右上方的 Offset ( +，—)来为所有边添加阴影。
