GraalVM 是一个多语言运行时。它可以运行用许多语言编写的应用程序——JVM 语言如 Java、Scala、Groovy 和 Kotlin，以及非 JVM 语言如 Python、R、Ruby、JavaScript 和 LLVM。GraalVM 还允许任何这些语言的应用程序执行用它支持的任何其他语言编写的代码片段。这就像出租车司机会说多种语言，当你开始插话时，他也完全能够理解你…
