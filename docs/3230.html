Python中的Self有什么用？

self用于表示类的实例。使用这个关键字，您可以在python中访问类的属性和方法。它将属性与给定的参数绑定在一起。我们使用self的原因是Python不使用' @ '语法来引用实例属性。在Python中，我们有一些方法可以自动传递实例，但不能自动接收。

举例:

class food():
# init method or constructor
def __init__(self, fruit, color):
self.fruit = fruit
self.color = colordef show(self):
print("fruit is", self.fruit)
print("color is", self.color )apple = food("apple", "red")
grapes = food("grapes", "green")apple.show()
grapes.show()

输出:

Fruit is apple
color is red
Fruit is grapes
color is green

Python类自构造函数

self也用于指类中的变量字段。让我们举个例子，看看它是如何工作的:

# name made in constructor
def __init__(self, John):
self.name = Johndef get_person_name(self):
return self.name

在上面的例子中，self指的是整个Person类的name变量。在这里，如果我们在一个方法中有一个变量，self将不起作用。该变量只在该方法运行时存在，因此是该方法的局部变量。为了定义整个类的全局字段或变量，我们需要在类方法之外定义它们。

自我是关键词吗？

self用于不同的地方，通常被认为是一个关键字。但与C++不同的是，self在Python中并不是一个关键字。

self是函数中的一个参数，用户可以使用不同的参数名来代替它。虽然建议使用self，因为它增加了代码的可读性。
