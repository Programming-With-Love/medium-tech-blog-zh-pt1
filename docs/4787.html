程序员字典:形参与实参，类型形参与类型实参

参数和自变量经常被混淆，但它们是完全不同的概念。我们来讨论一下它们是什么，有什么区别。我们也会明白什么是类型参数和类型实参。

参数vs变元

参数是在函数定义中定义的变量，而自变量是传递给函数的实际值。为了理解其中的区别，我们先来看一个例子函数及其用法:

fun randomString(length: Int): String {
    // ....
}randomString(10)

在这个例子中length是一个参数，而10(用于函数调用)是一个参数。以下是常见的定义:

参数是在函数声明中定义的变量。参数是传递给函数的该变量的实际值。

对话示例:

  • randomString 功能抛出错误。你传递给length 参数的参数是什么？
  • 是-1
  • 这就是原因。randomString没有为length 参数的负值做准备！

类型形参与类型实参

形参和实参的区别是通用的，它适用于所有类型的函数 : 方法，构造函数等。它也适用于通用的措辞。让我们讨论一下示例泛型类:

class Box<T>val a: Box<Int> = Box()

这里的Box是泛型类，定义了T 类型参数。在使用中，我们将Int指定为类型实参(此处见为什么是类型，而不是类)。常见定义如下:

类型参数 是泛型中声明的类型的蓝图或占位符。 类型参数 是用于参数化泛型的实际类型。
