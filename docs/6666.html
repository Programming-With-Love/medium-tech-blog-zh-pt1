Java 13将于2019年9月17日发布。除了约2300个错误修复和小的增强，新版本的Java包含5个主要增强，也称为JEPs (Java增强建议)。让我们仔细看看这些主要更新:文本块，开关表达式，重新实现了传统的套接字API，更新ZGC和动态CDS档案。
