Next.js 是非常流行的服务器端渲染 react 框架。这是非常有用的，但我们应该如何用 SCSS 或萨斯风格呢？

使用 next.js 创建应用程序后，可以使用与 create-next-app 相同的模板，并在 js 文件中编写自己的 css。
