我们很乐意让你尝试我们的 API ，所以Pinterest工程女性的一个小团队组织了一些有趣的挑战！欢迎来到#1014号展位参加挑战赛，了解您可以利用Pinterest API做些什么。完成挑战后，您可以在各种Pinterest板上使用您的解决方案代码，或者探索其他API端点。一旦您提交了您的解决方案，就可以在展台领取您的特别奖，并分享关于使用API的其他方法的想法。
