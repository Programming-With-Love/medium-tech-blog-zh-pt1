函数是 Javascript 中的一等公民。函数是为了可重用性而编写代码的地方。当你在函数中编写代码时，代码要简洁得多，并且很容易测试和理解。在 Javascript 中，函数是一个对象，所以它可以用作任何其他值，可以返回或传递，等等。
