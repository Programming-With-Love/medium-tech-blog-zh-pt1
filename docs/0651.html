游戏开发库

AGDK包括解决Android游戏开发痛点的 C和C++库。使用 GameActivity 以及我们的文本输入和游戏控制器库，你可以编写你的游戏循环，而不必接触JNI代码。当您将它与我们现有的双簧管音频库和帧调步库相结合时，我们已经构建了高性能的方法来获得您游戏所需的关键服务。

游戏开发扩展

AGDK的另一部分是针对Visual Studio 的 Android游戏开发扩展，它现在对所有开发者开放。它允许您在现有的Visual C++游戏项目中添加Android作为目标平台。您可以开发C /C++多平台游戏，部署到Android设备或仿真器上并在其上进行调试，所有这些都在一个IDE中使用一组项目文件完成。该扩展支持最新版本的Android SDK和NDK，并与我们最流行的工具配合使用，如Android Studio profilers、logcat、SDK管理器和虚拟设备管理器。

优化您的游戏

最后，AGDK包括工具和库来帮助你进一步优化你的游戏。在库的前端， Android性能调优器提供了用户遥测功能——现在能够跟踪加载时间。 Android GPU Inspector beta 现在支持帧剖析，允许你捕捉应用或游戏的单帧，检查Vulkan API的整个状态以及来自GPU的剖析数据。它还通过使用ANGLE在Vulkan API上运行OpenGL工作负载来支持OpenGL中的概要分析。

游戏用Android 12🎮

Android 12包括下载时的游戏，让你的用户更快地进入游戏。这意味着一些游戏资产不是在安装时下载的，而是在初始安装后在后台下载的。如果你使用App Bundle，这一切都不需要你对你的游戏做任何改动。

Android 12还引入了游戏模式API，允许游戏玩家选择性能配置文件，如更好的电池寿命或性能模式。API允许Android交流玩家的偏好，这样你就可以相应地调整你的游戏，但系统也可以代表他们干预自动调整游戏的游戏模式干预。你可以通过一个明显的设置选择退出这些干预，我们已经给提供了一个表格，所以你可以帮助我们为你的游戏调整这些干预。

安卓12 beta 3🤖

我们发布了Android 12 的第三个测试版，其中包括API 31 的最终API以及一些新功能。例如，我们引入了一个新的 ScrollCapture API 来帮助支持非常受欢迎的滚动屏幕截图功能。

我们还添加了新的隐私指示器API到 WindowInsets ，让你可以获得指示器的最大界限及其在屏幕上的相对位置，以及新的权限，允许与配套设备管理器配对的应用程序启动前台服务。

AppSearch Jetpack库允许您在Android 4.0+上以本地存储模式在您的应用内使用AppSearch，并将在平台存储模式中支持Android 12+的中央索引，这允许系统在系统UI表面和其他应用内显示您的应用的数据。
