我教 Java 已经有一段时间了。所以我经常分享如何更快更好的学习编程的建议。在与我的学生交谈时，我发现有一件事让他们非常分心:编码焦虑。这真是个麻烦。有些人发现很难集中注意力超过一个小时。有些人因为找工作压力过大。
