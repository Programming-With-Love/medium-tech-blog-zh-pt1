如果你熟悉Square是如何制作SDK的，那么你就会知道，要了解我们最新SDK版本的新特性，最好的方法之一就是查看我们在github上的API规范的差异(链接)。

让我们深入了解每一项变化:

订单支持

此版本中最大的更新是对订单端点的支持。新的订单端点允许您在线创建明细交易，是在线销售报告的一大改进。你可以在官方公告中阅读更多内容，或者直接进入文档。

错误修复

在任何新版本中，总会有一些早期的错误被修复。这一项包括以下内容:

  • 修正了 Python SDK readme中的一个错别字。
  • 改进了几个模型属性的描述，比如BatchUpsertCatalogObjectsRequest的断开链接。
  • 为一些属性添加了一个maxLength描述。
