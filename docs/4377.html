谷歌开发者专家 是一个充满激情的开发者社区，他们喜欢与他人分享他们的知识。他们中的许多人专门研究机器学习。

以下是展示上季度ML GDEs成就的一些亮点，这些成就对全球ML生态系统做出了贡献。如果你有兴趣成为ML GDE，请向下滚动，看看你如何申请！

ML开发者大会@Google I/O

在今年的I/O大会上，我们举办了两次ML开发者聚会(美国/APAC 和 EMEA/APAC )。梅尔夫·诺扬 / 优素福·萨格斯(土耳其)萨亚克·保罗 / 巴维什·巴特(印度)利·强生 / 玛格利特·梅纳德-里德(美国)大卫·卡多佐(哥伦比亚)维尼西乌斯·卡里达 / 阿纳尔多·瓜尔伯特

I/O扩展2021

在I/O之后，许多ML GDEs在他们的博客上发布了I/O的摘要。 Chansung Park (韩国)概述了 ML主题演讲摘要，而美国 Victor Dibia 总结了来自Google IO 2021 的十大机器学习和设计见解。

Vertex AI是活动中的话题。来自日本的Minori松田隼写了一篇日本文章，标题是“介绍强大的顶点人工智能汽车预测”。同样， Piero Esposito (巴西)发布了一篇名为“Vertex AI的无服务器机器学习管道:简介”的文章，其中包括一篇关于完全定制代码的教程。总部位于印度的 Sayak Paul 在Vertex AI发布后不久与人合作撰写了一篇博文讨论Vertex AI中的关键部分，展示了如何使用Vertex AI运行TensorFlow培训工作。

诸如Google Developers Groups(GDG)和tensor flow users Groups(TFUG)等社区举办了扩展活动，在这些活动中，演讲者进一步讨论了来自I/O的不同ML主题，包括来自中国的宋林对来自I/O的 TensorFlow亮点和应用体验的演讲，该演讲有24k在线参与者， Chansung Park (韩国)还就什么是顶点AI以及您可以用顶点AI做什么进行了演讲。

云人工智能

Leigh Johnson (美国)写了一篇详细的文章以solo创始人的身份软发布了一款AI/ML产品涵盖了GCP AutoML Vision、GCP IoT Core、TensorFlow Model Garden和TensorFlow.js。这是一个Solo创始人开发用于检测3D打印机打印故障的ML产品的故事(关于这个惊人故事的更多内容即将推出，敬请关注！)

演示和代码示例来自维克多·迪比亚(美国)纽约州出租车项目、米诺里·松田隼(日本)关于AutoML和AI平台笔记本的文章、 Srivatsan Srinivasan (美国)的视频教程、 Sayak Paul (印度)的tensor flow与AI平台的分布式培训& Docker

Aqsa Kausar (巴基斯坦)在国际妇女节菲律宾活动上做了一个关于谷歌云端可解释人工智能的演讲。她解释了它的重要性，以及它在ML工作流程中的应用位置和方式。

最后， ML Lab ，来自尼日利亚的 Robert John 介绍了GCP的ML景观，涵盖从BigQueryML到AutoML到TensorFlow和AI平台。

张量流

Eliyar Eziz (中国)出版了一本书“ TensorFlow 2，带有现实生活中的用例”。来自美国的Gant Laborde 撰写了 O'reilly的书《学习tensor flow . js》并写了一篇文章“没有数据没有问题——tensor flow . js迁移学习”，内容是关于寻找新的数据集，在以前没有模型训练过的地方大胆地进行训练。他还出版了“一个Riddikulus数据集”，谈论创建哈利波特数据集。

总部位于香港的王官发表了一篇研究论文“迭代扩张卷积神经网络分词”，涵盖了Keras在TensorFlow上实现的最先进的性能改进。

突尼斯的埃利斯·马奈写了一篇文章“成为Tensorflow认证开发者”，介绍TensorFlow认证指南和技巧。

总部位于希腊的乔治·索卢皮斯写了一篇教程“使用Colab TPU 微调BERT模型”，讲述如何使用Colab的TPU(v2–8)微调一个经过希腊语专门训练的BERT模型，以执行文本分类的下游任务。

JAX

总部位于印度的Aakash Nain已经发布了TF-JAX教程系列(第一部分、第二部分、第三部分、第四部分)，旨在向所有人传授TensorFlow和JAX框架的构建模块。

来自台湾的魏哲仁的在线聚会 Tensorflow和JAX 涵盖了JAX的介绍和使用案例。它还涉及了写张量流模型和训练循环的不同方法。

YouTube视频神经网络，有一个在JAX写的实际例子，可能是第一个葡萄牙语的JAX techtalk，作者是若昂·古伊列梅·马代亚·阿劳若(巴西)。

卡格尔

笔记本“带有句子嵌入的简单贝叶斯岭”作者埃尔图鲁尔·德米尔(土耳其)讲述了一项自然语言处理任务，该任务使用伯特微调，然后在变形金刚生成的句子嵌入之上进行简单线性回归。

来自韩国的韩优·李做了一个关于“学习机器学习和tensorflow与Kaggle竞赛”的演讲。他解释了如何使用Kaggle平台学习ML。

研究

机器学习和深度学习研究的进展正在改变我们的技术，许多ML GDEs对此感兴趣并做出了贡献。

Karim Beguir (英国)与DeepMind团队合作撰写了一篇论文，内容涉及一种使用深度强化学习解决机器人操纵任务的新颖组合方法。它在NeurIPS研讨会上被接受。

最后，来自印度的 Sayak Paul 与Pin-陈愉一起发表了一篇研究论文，“视觉变形器是健壮的学习者”，涵盖了视觉变形器(ViT)对常见的腐败和干扰、分布变化和自然对抗例子的鲁棒性。
